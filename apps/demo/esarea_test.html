Test page for esarea

Textarea with data-esarea-disabled="true"

  • When you are using esarea <= 0.0.20, you will see strange behaviors due to the conflict between esarea and esalike.
  • When you are using esarea > 0.0.20 or any version which ignores textarea with 'data-esarea-disabled="true"', only esalike should properly work.

Textarea without data-esarea-disabled="true"

  • Any version of esarea should work because esalike will not be applied for textarea elements those don't have the 'data-esarea-disabled="true"' attribute.
